# 图片目录说明

这个目录用于存放网站的图片资源。

## 图片命名规范

### Logo图片
- `logo.png` - 网站Logo (建议尺寸: 100x70)

### Banner图片
- `banner.jpg` - 首页Banner (建议尺寸: 1900x430)

### 关于我们图片
- `about-us.jpg` - 关于我们图片 (建议尺寸: 200x150)

### 产品图片
- `product-1.jpg` - 产品1图片
- `product-2.jpg` - 产品2图片
- `product-3.jpg` - 产品3图片
- `product-4.jpg` - 产品4图片
- (建议尺寸: 224x224)

## 使用方法

1. 将你的图片放到这个目录
2. 确保图片文件名与上面的命名一致
3. 图片会自动在网站中显示

## 支持的图片格式

- .jpg / .jpeg
- .png
- .gif
- .webp

## 注意事项

- 建议使用优化后的图片，避免文件过大
- 产品图片建议使用正方形
- Banner图片建议使用1900x430比例
